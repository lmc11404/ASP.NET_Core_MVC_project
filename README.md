# 我ㄉ大專 我真蚌
# 我超蚌
# 考試都考億百分
# 超長笑話

有一天，一隻鴿子、豬和一隻猴子決定去參加一個聚會。

### 第一天：
鴿子一早就到達了聚會現場，牠很高興地坐在一個大沙發上，開始吃起了聚會的點心。牠是個非常外向的鴿子，總是和周圍的人打招呼。

豬在後來到了，牠走進聚會，顯得有點不自在。豬很小心翼翼地站在一角，什麼也不做，只是看著其他動物。牠其實是一隻有點害羞的豬，雖然牠很喜歡吃，但牠通常不太參與社交活動。

猴子最晚才到達，牠在門口快速轉了一圈，翻著跟在牠後面的香蕉。猴子看到其他動物都已經在聊天，牠立刻跳上桌子，開始展示自己的雜技。牠一邊表演一邊笑著說：“看我，這是我為這個聚會準備的特技！你們要不要學？”

### 第三天：
聚會進行到第三天，鴿子開始感覺有點無聊。牠決定換個方式娛樂自己，牠拿出了一把小提琴，開始演奏。豬聽到音樂後，終於放下了自己的戒心，走到鴿子旁邊開始跳舞。豬的舞蹈有點笨拙，但卻非常有趣。

猴子看到豬跳舞，也興奮地跳了起來，但猴子的舞蹈更具創意，牠一邊跳舞一邊摔倒，還模仿鴿子拉小提琴的動作。這樣的表演逗得大家哈哈大笑，連其他動物也來加入，場面變得非常熱鬧。

### 第五天：
聚會進入了第五天，豬開始覺得不再那麼害羞了。牠不僅和其他動物交談，還主動提出了一個點子：每個動物都應該為聚會上的每個人做一件事來表達感謝。鴿子提議牠可以教大家怎麼拉小提琴，猴子則決定教大家如何做雜技，而豬則決定展示如何運用鼻子來挖掘土地。

### 最後：
在聚會的最後一天，三隻動物都從彼此身上學到了很多。鴿子學會了如何放鬆，豬學會了如何打開自己與他人交往，猴子則學會了如何更有耐心地對待其他動物。

這次聚會，雖然長得讓人有點累，但每一刻都充滿了歡笑與友情。

**結尾：**  
有一天，一位路過的獅子看到了這些動物，牠驚訝地問：“你們三個是怎麼在一起的？”

鴿子微笑著回答：“我們學會了，不管你是誰，做自己，與朋友一起，是最重要的。”

獅子聽後微微一笑，說：“我以為你們會告訴我怎麼跳舞的！”
